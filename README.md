# 凿一面墙 · 科技安全

网络25303班第八组翻转课堂展示项目。  
本项目以“中国科技安全”为主题，采用网页 PPT 的形式呈现，共 17 页，包含主题演示页面与配套讲稿。

## 项目内容
 
- `index.html`：网页 PPT 主文件
- `jianggao.txt`：展示讲稿参考
- `images/`：演示所用图片素材
- `assets/`：页面运行所需资源文件

## 运行方式

下载仓库后，双击 `index.html`,使用浏览器打开即可观看。  
演示时可使用键盘方向键进行翻页。
